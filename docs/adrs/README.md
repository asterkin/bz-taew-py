# Architecture Decision Records (ADRs) – Python Implementation

This directory contains architecture decision records specific to the Python implementation of the BlueZone application.

For language-neutral and cross-platform architectural decisions, refer to the [shared ADRs in the common project](../../common/docs/adrs/README.md).

Python-specific ADRs capture decisions related to:

- Language-specific architectural adaptations.
- Tooling, packaging, and dependency management.
- Testing strategies, deployment methods, and runtime environments.
